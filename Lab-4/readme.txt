## MATLAB 文件

* `main.m` - main 的 MATLAB 代码文件
* `fc.m` - 用于前馈计算的 MATLAB 代码文件。
* `bc.m` - 用于反向计算的 MATLAB 代码文件。
* 'cost.m' - 成本函数的 MATLAB 代码文件。
* 'accuracy.m' - 精度的 MATLAB 代码文件（评估索引）。
* `mnist_small_matlab.mat` - 手写数字数据集的 MATLAB 数据文件

## Python 文件

* `main.py` - main 的 PAYTHON 代码文件
* `utils.py` - utils 的 PAYTHON 代码文件，例如fc, bc, 成本, 准确性
* `mnist_small_matlab.mat` - 手写数字数据集的MATLAB数据文件（它可以被python读取！）
