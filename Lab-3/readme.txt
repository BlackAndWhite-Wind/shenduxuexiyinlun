## 文件

* `bp_online.m` - 在线 BP 算法的 MATLAB 代码文件，带有用于评估的异或蠕虫示例
* `bp_batch.m` - 批处理 BP 算法的 MATLAB 代码文件，带有用于评估的异或蠕虫示例
* `fc.m` - 用于前馈计算的 MATLAB 代码文件
* `bc.m` - 用于反向计算的 MATLAB 代码文件

## 任务提示

### 任务 0：实现前馈和后向计算
1.在`fc.m`中，实现前向计算（以分量或向量形式），同时返回激活和网络输入
2.在`bc.m`中，实现向后计算（分量或向量形式）

### 任务 1：实现在线BP算法
在`bp_online.m`中：
1. 计算激活`a1`、`a2`、`a3`和净输入`z2`、`z3`
2. 计算成本函数`J`
3.计算灵敏度`delta3`，`delta2`
4. 计算关于权重`dw1`、`dw2`的梯度
5. 更新权重`w1`、`w2`

### 任务 2：实现批量 BP 算法
在`bp_batch.m`中：
1. 计算激活`a1`、`a2`、`a3`和净输入`z2`、`z3`
2. 计算成本函数`J`
3.计算灵敏度`delta3`，`delta2`
4. 累积关于权重`dw1`、`dw2`的梯度
5. 更新权重`w1`、`w2`
